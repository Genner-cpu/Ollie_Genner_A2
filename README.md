**SmartTank** is a Python-based application built with **Pygame** that allows users to calculate and visualise vehicle data such as fuel efficiency, range, tank usage, tank size, and trip cost.  
Developed as part of the *Working with Data and Code* unit, this project demonstrates the use of data-driven calculations and simple interactive graphics.

---

Follow these steps to run *SmartTank* on your computer:

1. **Download or clone** this repository.
2. Ensure **Python 3.x** and **Pygame** are installed.
3. Ensure all files are in the same folder as the Pygame file
4. use fn f5 to open the application
